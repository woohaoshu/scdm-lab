运行项目	npm run start == ng serve -o	（-o 启动Index页面 --prod -- aot）

复制到centos	pscp -P 443 G:\angular\scdm\dist\scdm.zip shuwsc@dmis.shu.edu.cn:/home/shuwsc

unzip解压	sudo unzip scdm.zip
重启nginx	sudo systemctl reload nginx
